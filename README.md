# Federated Anomaly Detector

A system for detecting and analyzing anomalies in distributed systems using federated learning approaches.

## Overview

This project implements a federated learning framework for training anomaly detection models across distributed data sources without sharing raw data. It's designed for scenarios where data privacy, bandwidth limitations, or regulatory constraints prevent centralizing sensitive data.

## Features

- Federated learning orchestration for distributed training
- Multiple anomaly detection algorithms (Isolation Forest, Autoencoder, Statistical Methods)
- Secure aggregation of model updates
- Privacy-preserving techniques
- Real-time anomaly detection
- Visualization and reporting tools

## Installation

```bash
git clone https://github.com/Luv-Goel/federated-anomaly-detector
cd federated-anomaly-detector
pip install -r requirements.txt
```
